== Description ==


"Tips, Gratuity, Donations, Fees For WooCommerce" is a plugin for allowing shop customers to tip/donate certain amount as per their choice during shopping from store.

= Plugin Features =

* Tip/Donation During checkout.
* Tip/Donation amount taxable enable/disable.
* Default amount add.
* Quick amount display.
* Quick amount enable/disable.
* Display for cart/checkout or both.

= Features Location =

* Main plugin page - Woocommerce -> Tip/Donation


NOTE: This plugin require atleast Woocommerce v2.5.0.

== Installation ==

1. Upload the plugin.
2. Activate plugin
3. Go to Woocommerce -> Tip/Donation.
4. Configure or Use Default Options
